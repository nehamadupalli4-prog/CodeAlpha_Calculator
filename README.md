# CodeAlpha Calculator

A modern and responsive calculator application developed as part of the CodeAlpha Frontend Development Internship. This project performs basic arithmetic calculations with a clean user interface and keyboard support.

## 📌 Project Overview

The Calculator application allows users to perform basic mathematical operations such as addition, subtraction, multiplication, and division. The project is designed with a responsive layout to ensure a smooth experience across desktop and mobile devices.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear and Reset functionality
* Real-time calculation display
* Keyboard support
* Responsive design
* User-friendly interface

## 🛠️ Technologies Used

* React
* TypeScript
* Vite
* CSS
* JavaScript

## ⚙️ Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/nehamadupalli4-prog/CodeAlpha_Calculator.git
```

### Navigate to the Project Directory

```bash
cd CodeAlpha_Calculator
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
